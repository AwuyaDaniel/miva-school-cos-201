# Student Record System

A simple C program that lets you manage student records.

## What it does

- Add, view, edit, and delete students
- Search for a student by roll number
- Calculate average marks
- Sort students by marks
- Save and load records from a file

## Files

- `main.c` - Runs the menu
- `student.c` - Contains all the functions
- `student.h` - Header file with the Student structure

## How to run

```bash
gcc main.c student.c -o student_system
./student_system
```

## Requirements

- A C compiler (like GCC)
